# Amaç: İklim değişikliğini önleyecek bir proje.

Proje Amacı:
Bu discord botumuz iklim değişikliği hakkında bize bilgi vericek ve  nasıl önleyebileceğimize dair bize öneri vericek.

## Nasıl Çalışır?

Bota komut girerek,bize iklim değişikliği hakkında bilgi vericek.En sonunda da önlemek için neler yapmamız gerektiğini söyliyecek.

###  Model'de hangi uygulamalar kullanıldı?:

Modelimi discord'u ve visual studio code'u kullanarak oluşturacağım.
